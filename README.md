This is the code for the manuscript "Integrated QTL mapping and CRISPR screening in pooled iPSC-derived
microglia reveals genetic drivers of neurodegenerative risk". 
All scripts were run in Ubuntu with R v.4.4.1 or Python v.3.10.10.

BiorXiv version: https://www.biorxiv.org/content/10.1101/2025.08.18.670767v1
The latest version of the zenodo repository is: https://zenodo.org/records/20689990
The PoodleR repository to estimate donor proportions from pooled experiments with WGS is provided separately here: https://github.com/TrynkaLab/poodleR
